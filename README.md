# Selveo — sitio web

Sitio hecho con Next.js. Instrucciones completas de uso en el mensaje que Claude te dio al entregar el proyecto. Resumen rápido:

- Correr en tu computadora: `npm install` y luego `npm run dev` (abre http://localhost:3000)
- Panel oculto: http://localhost:3000/panel-selveo
- Contraseña del panel: se cambia en el archivo `.env.local`, variable `ADMIN_PASSWORD`
- Propiedades: se guardan en `data/properties.json` (local) o en Vercel Blob (producción)
- Publicar gratis: subir este proyecto a GitHub y conectarlo en vercel.com
